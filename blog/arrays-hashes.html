Ruby Arrays & Hashes

10.2.15

Arrays

Put simply, arrays are ordered collections of objects. Each collection can hold multiple elements of any data type, including other arrays. Each element can be referred to by an index. An important thing to keep in mind is that arrays start at the zeroth index. The last item in an array can be referred to by the -1 index (accordingly, the penultimate array item would be indexed at -2). There is more than one way to create an array. For example...

Creation with square brackets:

cheese = ["brie", "gruyere", "swiss"]
	#cheese[0] cheese[1] cheese[2]
	#The respective index positions.
Creation with the Array.new method, which can also take arguments and a block (argument and block not depicted here):
cheese = Array.new
cheese.push "brie"
cheese.push "gruyere"
cheese.push "swiss"
Both arrays we created above will give you the same output:
puts cheese
=> brie
   gruyere
   swiss
***To learn about all the cool methods you can apply to arrays, check out the Ruby documents here.***

Hashes

A hash can be thought of as a database or a collection that contains key value pairings. Keys are unique, and each key is to be assigned a value. They're also known as associative arrays due to their similarities to arrays, but they differ in that arrays index content by integers, while hashes allow content to be indexed using any object type. Lets take a look at a couple ways we can create a hash...

Creation with curly braces:

#Option 1 formatting:
cheese = {"brie": "mildly tangy and fruity", "gruyere": "creamy and sweet", "swiss": "slightly nutty and mild"}

#Option 2 formatting:
cheese = {"brie" => "mildly tangy and fruity", "gruyere" => "creamy and sweet", "swiss" => "slightly nutty and mild"}
Creation with the Hash.new method:
cheese = Hash.new
cheese["brie"] = "mildly tangy and fruity"
cheese["gruyere"] = "creamy and sweet"
cheese["swiss"] = "slightly nutty and mild"
Both methods will serve up the same output:
puts cheese
=> {"brie"=>"mildly tangy and fruity", "gruyere"=>"creamy and sweet", "swiss"=>"slightly nutty and mild"}
Each key you named in effect became the "hash index" that you can call to access your value:
cheese["brie"]
=> "mildly tangy and fruity"
Rather than calling your value one key at a time, there is a nifty hash method that will return all your values in an array:
cheese.values
=> ["mildly tangy and fruity", "creamy and sweet", "slightly nutty and mild"]
***There are many other hash methods you can explore in the Ruby documents here at your leisure.***
